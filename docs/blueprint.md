# **App Name**: EventHub+

## Core Features:

- Event Discovery: Browse and search for events based on interests and tags.
- AI Recommendations: Get personalized event recommendations using EventBuddy AI.
- QR Attendance: Check in to events using QR code scanning.
- Certificate Vault: Store and manage event certificates.
- Automated Report Generation: Generate event reports including attendance, photos with geolocation, and procurement data.
- Skill Gap Analysis: Analyze student event engagement data to identify skill gaps and inform curriculum planning.
- Suggested Sections to Invite: As a tool, recommend class sections to invite based on timetable data to improve event attendance.

## Style Guidelines:

- Primary color: Deep blue (#293B7B) to evoke trust and professionalism, fitting for an academic environment.
- Background color: Very light blue (#E0E5F2) for a clean and calming user interface.
- Accent color: Soft violet (#7B61FF) to complement the primary color and add a touch of modernity and innovation.
- Body and headline font: 'Inter' sans-serif for a modern, neutral, readable style.
- Use a consistent set of minimalist icons to represent different event categories and actions.
- A clean and organized layout with clear sections for event listings, recommendations, and user profile information.
- Subtle transitions and animations to enhance user experience and provide feedback during interactions.
